# owlapi_wrapper

[![build-test](https://github.com/alexskr/owlapi_wrapper/actions/workflows/maven.yml/badge.svg)](https://github.com/alexskr/owlapi_wrapper/actions/workflows/maven.yml)


A command line utility that wraps the Java [OWL API](https://github.com/owlcs/owlapi) for parsing RDFS, OWL, and OBO ontologies.
